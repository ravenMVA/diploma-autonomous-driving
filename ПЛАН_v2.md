## Новая концепция

### Суть задачи

Агент наблюдает изображение с камеры и должен **удержаться на полосе**.

- **Классификатор (чёрный ящик):** нейросеть оценивает, насколько автомобиль отклонился от центра полосы — выдаёт ошибку отклонения `e ∈ [-1, 1]`
- **ПИД-регулятор (агент влияния):** получает ошибку от нейросети и вычисляет корректирующий сигнал руля
- **Обратная связь:** если `|e| < порог` → агент в полосе (выигрыш +1), иначе — вне полосы (проигрыш −1)
- **Цель:** минимизировать интегральную ошибку `∫|e(t)| dt`

### Схема пайплайна

```
Изображение камеры
        │
        ▼
┌───────────────────┐
│  Нейросеть (CNN)  │  ← "чёрный ящик"
│  Классификатор    │    обучается на размеченных данных
│  отклонения       │    выход: e ∈ [-1, 1]
└───────────────────┘
        │
        ▼
     ошибка e
        │
        ▼
┌───────────────────┐
│  ПИД-регулятор   │  ← агент влияния на чёрный ящик
│  u = Kp·e        │
│    + Ki·∫e dt    │
│    + Kd·(de/dt)  │
└───────────────────┘
        │
        ▼
   управляющий сигнал u (угол руля)
        │
        ▼
┌──────────────────────────┐
│  Классификатор полосы    │
│  |u| < порог → В ПОЛОСЕ  │  → награда +1
│  |u| ≥ порог → ВНЕ ПОЛОСЫ│  → штраф  −1
└──────────────────────────┘
```

```

```

**Изображения:**
- Размер: 32×100 (было 66×200) — в 4× меньше пикселей
- Цветовое пространство: grayscale или YUV V-канал

## Архитектура модели (лёгкая)

**Вход:** 1×32×100 (grayscale)
**Выход:** скаляр `e ∈ [-1, 1]` (отклонение от центра полосы)

```
Conv 3×3 stride 2 → 8 фильтров  + ReLU              (выход: 8×16×50)
MaxPool 2×2                                           (выход: 8×8×25)
Conv 3×3 stride 1 → 16 фильтров + ReLU              (выход: 16×6×23)
MaxPool 2×2                                           (выход: 16×3×11)
Flatten → 528
Dense 64 → ReLU
Dense 1  → Tanh  (выход: e ∈ [-1, 1])
```

**Параметры:** ~37k (против 252k в PilotNet)
**Потери:** MSE по углу руля (непрерывный таргет), порог для бинаризации — постфактум

## ПИД-регулятор

```python
class PIDController:
    def __init__(self, Kp=0.8, Ki=0.01, Kd=0.15):
        self.Kp, self.Ki, self.Kd = Kp, Ki, Kd
        self.integral = 0.0
        self.prev_error = 0.0

    def step(self, error, dt=1/30):
        self.integral += error * dt
        derivative = (error - self.prev_error) / dt
        self.prev_error = error
        u = self.Kp * error + self.Ki * self.integral + self.Kd * derivative
        return float(np.clip(u, -1.0, 1.0))

    def reset(self):
        self.integral = 0.0
        self.prev_error = 0.0
```

**Подбор коэффициентов:**
- Ручной: Kp методом Ziegler-Nichols (критическое усиление)
- Автоматический: grid search по минимуму `∫|e| dt` на валидационном треке

## Агент и функция вознаграждения

```python
THRESHOLD = 0.15  # граница "в полосе"

def get_reward(error):
    """Бинарная классификация: в полосе или нет."""
    if abs(error) < THRESHOLD:
        return +1.0   # в полосе — выигрыш
    else:
        return -1.0   # вне полосы — проигрыш

def run_episode(frames, model, pid):
    pid.reset()
    total_reward = 0
    errors = []

    for frame in frames:
        e = model.predict(frame)          # нейросеть (чёрный ящик)
        u = pid.step(e)                   # ПИД корректирует
        reward = get_reward(u)            # классификация
        total_reward += reward
        errors.append(abs(e))

    cte = np.mean(errors)  # cross-track error (интегральная ошибка)
    return total_reward, cte
```

3. Загрузка изображений → grayscale 32×100 → numpy массив → сохранение в `data/cache.npy`

3. Обучение 20 эпох (EarlyStopping patience=5)
4. График loss / accuracy
5. Сохранение `models/model_v2.pth`
6. Метрики на тесте: MSE, MAE, точность классификации (в/вне полосы)

### `03_agent_pid.ipynb` (5 мин)
1. Загрузка модели
2. Демонстрация на 200 последовательных кадрах из теста
3. График: предсказанная ошибка `e(t)` vs реальный угол руля
4. ПИД-коррекция: `u(t)` — управляющий сигнал
5. Сравнение: только модель vs модель + ПИД (интегральная ошибка)
6. График вознаграждений: +1/-1 по кадрам
7. Подбор коэффициентов Kp, Ki, Kd — grid search, тепловая карта

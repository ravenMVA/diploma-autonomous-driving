# Система автономного управления автомобилем на основе нейронных сетей

**Дипломная работа** | Behavioral Cloning | NVIDIA PilotNet CNN

---

## О проекте

Система обучается управлять автомобилем, имитируя действия человека-водителя.
На вход подаётся изображение с фронтальной камеры, на выход — угол поворота руля.

**Подход:** End-to-end обучение (Behavioral Cloning)
**Архитектура:** NVIDIA PilotNet (5 свёрточных + 4 полносвязных слоя)
**Датасет:** Udacity Self-Driving Car Dataset

## Описание модели

### Архитектура NVIDIA PilotNet

| Слой | Тип | Параметры |
|------|-----|-----------|
| 1 | Conv2D | 24 фильтра, 5×5, stride 2 |
| 2 | Conv2D | 36 фильтров, 5×5, stride 2 |
| 3 | Conv2D | 48 фильтров, 5×5, stride 2 |
| 4 | Conv2D | 64 фильтра, 3×3 |
| 5 | Conv2D | 64 фильтра, 3×3 |
| — | Flatten + Dropout | — |
| 6 | Dense | 100 нейронов |
| 7 | Dense | 50 нейронов |
| 8 | Dense | 10 нейронов |
| 9 | Dense | 1 выход (угол руля) |

**Вход:** изображение 66×200×3 (формат YUV)
**Выход:** угол поворота руля (значение от -1 до 1)

## Метрики качества

- **MSE** (Mean Squared Error) — основная функция потерь
- **MAE** (Mean Absolute Error) — средняя абсолютная ошибка в радианах
- **R²** (коэффициент детерминации) — насколько модель объясняет данные

## Датасет

**Udacity Self-Driving Car Dataset**
Источник: Kaggle — `ronamgir/udacity-car-dataset-simulator-challenge`

Содержит:
- Изображения с 3 камер (центральная, левая, правая)
- CSV с углами руля, скоростью, газом и тормозом
- ~45 000 кадров езды по трассе
